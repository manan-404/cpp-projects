# Market Management System

A console-based C++ project that simulates a simple market environment. Users can shop across three different stores — a Garment Shop, a Bakery, and a Bookstore — with a final combined bill displayed at the end.

---

## Features

- **Garment Shop** — Browse Ladies, Gents, and Children clothing across Party, Casual, and Sports wear categories
- **Bakery** — Choose from Biscuits, Snacks, Cakes, and Drinks with real-time bill tracking
- **Bookstore** — Browse and purchase books from Mathematics, English, and Programming categories
- **Combined Billing** — Tracks spending across all shops and displays a final market bill

---

## How to Run

### Using Dev C++
1. Open Dev C++
2. Open `market_management_system.cpp`
3. Press **F11** to Compile & Run

### Using Command Line (g++)

g++ market_management_system.cpp -o market
./market
